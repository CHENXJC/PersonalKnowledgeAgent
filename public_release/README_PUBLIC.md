# PersonalKnowledgeAgent Public Showcase Notes

PersonalKnowledgeAgent is a local-first personal knowledge base dashboard for
organizing sanitized Markdown notes about AI projects, learning, career planning,
business ideas, market observations, and long-term goals.

## Public Showcase Rule

The public version must use demo or sanitized notes only.

Do not publish the real private knowledge base. Do not publish private study
records, private career documents, personal identity documents, credentials,
account data, private logs, generated reports from real workflows, or raw local
files.

## What The Public Version Can Include

- Source code for the local dashboard.
- Public-safe documentation.
- Sanitized sample notes in `demo_knowledge_base/`.
- Tests.
- Screenshot instructions.
- Public release checklist.
- A generated inventory report only if it has been reviewed and contains no
  private paths or private file names.

## What The Public Version Must Exclude

- `.env` files.
- API keys, tokens, passwords, or credentials.
- Real private personal knowledge files.
- Passport, visa, bank, tax, medical, identity, or account recovery documents.
- Private logs or unreviewed generated outputs.
- Real emails, phone numbers, addresses, or customer data.

## Recommended Public Demo Flow

1. Run tests and compile checks.
2. Run the public release checker.
3. Review `demo_knowledge_base/` manually.
4. Capture screenshots using demo/sanitized content only.
5. Confirm `outputs/` contains no private report except `.gitkeep` or a reviewed
   non-sensitive report.
6. Only publish after all blocking risks are resolved.

## Current Mode

Public showcase preparation is in progress. The release mode is demo-only.
