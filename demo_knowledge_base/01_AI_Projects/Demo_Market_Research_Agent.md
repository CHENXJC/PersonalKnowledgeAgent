# Demo Market Research Agent

- Current checkpoint: DEMO-001
- Current stage: Sanitized public showcase sample
- Project type: Local-first research workflow demo
- Data status: Fictional sample content only

## Purpose

This fictional agent helps a small operator turn a broad market question into a
structured research brief, a assumptions list, and a set of follow-up questions.

## Demo Workflow

1. Collect a public-safe research question.
2. Break it into market, customer, competitor, and channel assumptions.
3. Produce a concise checklist for manual validation.

## Portfolio Value

- Demonstrates agent workflow design without requiring API keys.
- Shows how notes can preserve project decisions and next steps.
- Keeps all examples safe for public display.

## Safety Note

This is not based on a real customer or private market file.
