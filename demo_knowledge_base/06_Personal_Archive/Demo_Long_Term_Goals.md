# Demo Long Term Goals

## Theme

Build a stable personal learning system around projects, weekly review, and
public-safe documentation.

## Demo Goals

- Keep a weekly project review note.
- Separate private notes from public showcase notes.
- Build small tools that can be explained clearly.
- Review safety boundaries before publishing anything.

## Reflection Prompt

What information should stay private, and what can be safely turned into a
generic lesson or demo artifact?
