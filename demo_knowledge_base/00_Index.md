# Demo Knowledge Base Index

This folder contains sanitized demo notes for PersonalKnowledgeAgent public
showcase preparation. It is not a copy of the real private knowledge base.

## Demo Categories

- `01_AI_Projects`: Fictional AI project summaries.
- `02_University`: Generic learning notes.
- `03_Career`: Public-safe portfolio positioning notes.
- `04_Business_Ideas`: Fictional SME automation concepts.
- `05_Market_Investment`: Educational market observations only.
- `06_Personal_Archive`: Public-safe long-term goal notes.

## Safety Boundary

These notes do not contain real private study records, real resumes, accounts,
phone numbers, emails, addresses, API keys, tokens, passwords, identity
documents, bank details, visa information, or private local paths.
